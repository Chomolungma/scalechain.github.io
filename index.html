Problem

The number of Bitcoin transactions are growing over time, but Bitcoin full nodes require to keep all transactions from genesis block. The size of blockchain also growing over time. Increasing block size of Bitcoin is not an ultimate solution to this problem, as Bitcoin core still needs to keep all transactions data in a single node.

ScaleChain.io

ScaleChain.io focuses on solving the scalability issue of Bitcoin's blockchain. There are three ways to solve this issue. 1. Reduce database size for a node.
2. Reduce the transactions on blocks using side chains, and off-chain transactions.
